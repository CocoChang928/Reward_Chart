# 🏆 冒險蓋章挑戰！

專為小朋友設計的 **闯關式獎勵集章 App**！讓好習慣變得超有趣！  
100% 純前端、免安裝、免後端，所有資料存在瀏覽器 `localStorage`。

## ✨ 特色功能

| 功能 | 說明 |
|------|------|
| 🏰 **10 關闯關遊戲** | 每關 30 張印章，集滿後晉級下一關 |
| 🎨 **解鎖系統** | 每 10 張印章解鎖新大頭貼 + 新印章圖案 |
| ❗ **新內容通知** | 有新大頭貼可用時，大頭貼右上角出現紅色驚嘆號 |
| 💾 **匯出 / 匯入 JSON** | 備份資料防止遺失，重設前必備 |
| 👨‍👩‍👧‍👦 **多位小朋友** | 最多支援 3 位小朋友，各自獨立闯關 |
| 📱 **分享連結** | 產生 QR Code / 連結，小朋友可用平板查看進度（唯讀） |
| 📄 **匯出 PDF** | 匯出蓋章紀錄報告 |
| 🔊 **語音播報** | 蓋章時自動朗讀鼓勵語句（TTS） |
| 🎉 **慶祝動畫** | 里程碑和闯關成功都有動畫和紙花特效 |

## 🎮 關卡系統

- **10 關冒險**，每關不同主題：森林 → 海洋 → 高山 → 火山 → 冰雪 → 沙漠 → 月球 → 櫻花 → 雷霆 → 傳說殿堂
- **每關 30 格印章**，集滿自動晉級
- **累計每 10 個印章**（跨關卡）解鎖新的大頭貼和印章圖案
- **10 關全破**完成遊戲 🏆

## 🎨 素材資源

- **60 個印章圖案**：可愛動物、冒險道具、獎盃、魔法物品等
- **36 張大頭貼**：小熊系列、大猩猩(DK)系列、忍者貓、太空兔、機器人等

## 📖 使用方式

1. 用瀏覽器開啟 `index.html`
2. 點擊「⚙️ 家長設定」新增小朋友
3. 小朋友表現好 → 點空格 → 選印章 → 輸入原因 → 蓋章！
4. 集滿 30 格 → 闯關成功 → 挑戰下一關！

## 💾 資料備份

- **匯出 JSON**：點擊「💾 匯出」下載備份檔
- **匯入 JSON**：點擊「📂 匯入」讀取備份檔還原
- ⚠️ 重設前請務必先匯出備份！

## 📱 分享給小朋友

點擊「📱 分享」產生 QR Code 或連結，小朋友可以在平板/手機上查看進度（唯讀模式）。

## 🛠️ 技術

- 純 HTML / CSS / JavaScript（無框架）
- 資料存在 `localStorage`
- 分享透過 URL hash + LZ-String 壓縮
- PWA-ready 架構
